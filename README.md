

AI USAGE (Part2)
I used AI tools such as Gemini as a guide during development. I mainly used it to help me create a roadmap on how to approach the coding process and understand how to apply my professor’s feedback step by step.

I also asked for guidance in organizing the flow of the program, such as requesting a process or flowchart to better visualize the logic before implementing it. I avoided directly generating full code and instead used AI as a reference for structure and debugging.

Additionally, I used AI to generate simple cheat sheets for loops, methods, and classes to help me recall syntax and better understand how to apply them in my code.

All final decisions, logic, and implementation were done and reviewed by me.

During development, my project was initially connected to a different repository, so I manually transferred the final code to the correct repository before committing.
