Course & Year: BSIT - 2nd Year
Institution: PUP San Pedro Campus

Project Description
Jhear's Thrift Shop is a simple C# Console Application that functions as a basic sales system for an ukay-ukay business. The program displays a menu of 20 different fashion items with their respective prices and available stocks. Users can select a product by entering its ID and specifying the quantity they wish to buy. The system automatically calculates the total cost and applies a 10% discount for orders reaching PHP 5,000 or more. This project focuses on applying core programming concepts like arrays, loops, and basic object-oriented logic.
